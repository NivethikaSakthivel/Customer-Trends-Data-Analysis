# 📊 Customer Behavior Data Analysis using Python, SQL & Power BI

An end-to-end Data Analytics project that analyzes retail customer behavior using industry-standard tools and workflows. This project demonstrates how raw customer data can be transformed into meaningful business insights through data cleaning, exploratory analysis, SQL querying and interactive Power BI dashboards.

---

## 🚀 Project Objective

The objective of this project is to analyze customer shopping behavior and identify patterns that support business decision-making. The project follows a complete analytics pipeline, from preparing raw data to delivering insights through visual reports and dashboards.

---

## 🛠️ Technologies Used

* 🐍 Python
* 🗄️ SQL (MySQL/PostgreSQL)
* 📊 Power BI
* 📓 Jupyter Notebook
* 🐼 Pandas
* 📈 Matplotlib & Seaborn

---

## 📂 Project Workflow

### 1️⃣ Data Preparation & Cleaning (Python)

* Imported the retail customer dataset
* Performed data cleaning and preprocessing
* Handled missing values and inconsistent records
* Conducted Exploratory Data Analysis (EDA)
* Prepared the dataset for database storage

---

### 2️⃣ Database Creation & SQL Analysis

* Created a relational SQL database
* Loaded cleaned data into the database
* Executed SQL queries to answer business questions
* Analyzed:

  * Customer demographics
  * Purchasing behavior
  * Spending patterns
  * Loyalty trends
  * Product performance
  * Revenue insights

---

### 3️⃣ Interactive Dashboard (Power BI)

Designed a business intelligence dashboard featuring:

* Customer segmentation
* Sales overview
* Revenue analysis
* Purchase frequency
* Category-wise performance
* Customer spending distribution
* Interactive filters and slicers
* KPI cards and trend visualizations

---

### 4️⃣ Business Insights

The analysis helps answer key business questions such as:

* Which customer segments generate the highest revenue?
* Which products are purchased most frequently?
* What factors influence customer spending?
* Which age groups contribute the most to sales?
* How can customer retention be improved?
* What opportunities exist to increase business growth?
